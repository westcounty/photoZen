# Changelog

所有版本更新记录。

## [1.5.1] - 2026-01-15

### 新功能
- **保留照片列表排序**：新增排序功能，支持时间正序、时间倒序、随机排序

### 优化
- **快速分类横幅**：现在显示未添加标签的照片数量，而非所有保留照片数量
- **排序图标区分**：时间正序显示向上箭头，时间倒序显示向下箭头，随机显示随机图标
- **排序基准**：所有排序功能统一使用创建时间（dateAdded）而非修改时间
- **移动模式**：标签关联相册时，移动模式现在优先尝试直接移动文件，而非复制后删除

### 修复
- **复制照片时间戳**：修复复制照片后创建时间、修改时间、拍摄时间与原图不一致的严重问题，确保副本与原图时间完全一致便于正确排序

---

## [1.5.0] - 2026-01-15

### 新功能
- **照片排序**：筛选模式（Flow Sorter 和快速整理）新增排序功能，支持时间正序、时间倒序、随机排序三种方式，点击按钮即可切换
- **照片复制**：保留照片列表新增"复制照片"功能，可创建照片副本并完整保留所有 EXIF 信息和时间戳

### 优化
- **标签相册关联**：移动模式优化，优先使用系统移动 API 直接移动文件，避免复制后删除的低效操作
- **EXIF 保留**：复制照片时完整保留所有 EXIF 元数据（拍摄时间、GPS 位置、相机信息等）

### 修复
- 修复列表视图多选删除时可能导致的崩溃问题
- 批量操作改为原子性数据库操作，提升稳定性和性能
