# PhotoZen Logo 设计说明

> 版本: 2.0
> 更新日期: 2026-01-25
> 设计理念: 禅意取景框

---

## 设计理念

### 品牌内涵解析

```
PhotoZen = Photo (照片) + Zen (禅)

Photo 照片          Zen 禅意
├─ 捕捉瞬间        ├─ 简洁纯粹
├─ 珍贵记忆        ├─ 宁静专注
├─ 精选时刻        ├─ 有序和谐
└─ 视觉呈现        └─ 流动平衡
```

### 核心设计元素

新Logo融合三个核心意象：

1. **取景框** - 圆润的矩形框代表照片/相框，象征产品的核心功能
2. **禅意曲线** - 优雅的S形流线代表禅宗的流动与平衡感
3. **精选圆点** - 中心的圆点象征用户精选的珍贵照片

## 配色方案

### 主色调

| 色彩 | 色值 | 用途 |
|-----|------|------|
| 品牌青绿 | `#5EEAD4` | 主要图形元素 |
| 深邃蓝黑 | `#0A0A0F` | 背景主色 |
| 纯白高光 | `#FFFFFF` | 光泽细节 |

### 色彩语义

- **青绿色**: 代表专注、清新、现代科技感
- **深邃背景**: 优化照片观看体验，减少视觉干扰
- **微妙渐变**: 增添空间深度和高级质感

## 技术规格

### Adaptive Icon 结构

```
ic_launcher/
├── background (背景层)
│   ├─ 深邃蓝黑基色 #0A0A0F
│   ├─ 中心微光层
│   ├─ 顶部渐变
│   ├─ 底部深化
│   └─ 对角线质感纹理
│
├── foreground (前景层)
│   ├─ 取景框 (3.5px描边)
│   ├─ 主禅意曲线 (4px)
│   ├─ 辅助曲线 (2.5px, 60%透明)
│   ├─ 核心圆点 (12dp)
│   ├─ 高光点 (4dp)
│   ├─ 角落装饰 (4dp, 40%透明)
│   └─ 顶部光泽线
│
└── monochrome (单色层)
    └─ 简化版图形，适配Android 13+动态主题
```

### 尺寸规范

| 密度 | 尺寸 | 用途 |
|-----|------|------|
| mdpi | 48×48 dp | 基准尺寸 |
| hdpi | 72×72 px | 1.5x |
| xhdpi | 96×96 px | 2x |
| xxhdpi | 144×144 px | 3x |
| xxxhdpi | 192×192 px | 4x |

### 安全区域

Adaptive Icon 采用 108×108 dp 画布，核心图形位于中心 72×72 dp 安全区域内。

## 设计对比

### 旧版 vs 新版

| 维度 | 旧版 | 新版 |
|-----|------|------|
| 图形 | 通用镜头圆环 | 禅意取景框 |
| 辨识度 | 低，容易与相机App混淆 | 高，独特的品牌符号 |
| 品牌表达 | 仅体现"Photo" | 同时体现"Photo"+"Zen" |
| 层次感 | 扁平 | 丰富(多层透明度+光效) |
| 现代感 | 一般 | 精致现代 |

## 设计原则

### DO ✓
- 保持图标在深色背景上使用
- 确保安全区域内图形完整可见
- 使用指定的品牌色 #5EEAD4

### DON'T ✗
- 不要拉伸或压缩比例
- 不要更换品牌主色
- 不要在浅色背景上单独使用前景图形
- 不要添加额外的阴影或描边
